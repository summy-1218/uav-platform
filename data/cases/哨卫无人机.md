# 哨卫无人机总体设计总结报告

## 项目概述

*   **项目名称**：哨卫察打一体无人机系统
*   **设计团队**：北京航空航天大学学生团队（刘英伦，鲍禹冰，吴倩，姜皓严）
*   **设计理念**：以美国空军“忠诚僚机”概念为牵引，为我国南海及台湾海峡等特殊作战环境定向研发。
*   **项目目标**：开发一款具备短距起降、高空侦察、察打一体、信火一体、体系化作战能力的无人机。

## 设计流程与阶段

*   **SRR阶段 (系统需求评审)**：
    *   进行需求分析、作战环境调研（南海高温高湿高盐雾、大风）、同类型机型调研。
    *   明确了任务陈述、任务剖面、设计约束、初步重量估算。
    *   提出单发方案（方案一）与双发方案（方案二）草图，并进行初步权衡分析。

*   **SDR阶段 (系统设计评审)**：
    *   对需求参数、任务剖面、起飞重量进行修正。
    *   选定**双发布局、背负式进气、飞翼布局**作为最终方案方向。
    *   完成发动机初步选型、总体布局图绘制。
    *   开展了详细的岛栖环境适应性设计，制定了初步的抗腐蚀方案。

*   **CoDR阶段 (初步设计评审)**：
    *   完成详细的气动外形建模与迭代优化。
    *   确定了最终的三视图及关键尺寸参数。
    *   完善并确定了岛栖环境适应系统的具体设计方案。
    *   通过权衡分析，综合短距起降动力、隐身、高空性能等需求，最终确认**双发方案（方案二）**为最优方案。

## 最终方案详细描述

### 1. 总体布局与气动外形
*   **布局形式**：**飞翼布局**，带全动翼尖，采用小展弦比三角翼。
*   **动力布局**：**双发动机**，背负式进气，采用DSI鼓包以优化隐身和气动。
*   **关键几何参数**：
    *   前缘后掠角：56°
    *   后缘后掠角：5°
    *   机身长度：13.525m (经公式估算并修正)
    *   机身宽度(平均)：3m
    *   机身高度：2m
*   **推力矢量**：采用先进的**射流喷管技术**替代传统挡流板，提升操纵效率并减重。
*   **翼型**：基于高速作战需求，选用超临界翼型。

### 2. 任务能力与性能需求（概要）
*   **核心任务**：短距起降、高空侦察与监视、对地/对海打击。
*   **作战环境**：适应南海岛礁高温、高湿、高盐雾环境，具备抗大风干扰能力。
*   **关键性能**：短距起降能力、高隐身性（RCS低）、长航程/航时、高速巡航。

### 3. 岛栖环境适应系统设计
为确保在南海高盐雾、高温高湿环境下的可靠性和寿命，进行了专项抗腐蚀设计。

#### 3.1 材料选择方案
*   **主体结构材料**：航空铝合金 (7075-T6/2024-T3) 和钛合金 (Ti-6Al-4V)。
*   **工艺处理**：采用阳极氧化处理，搭配微弧氧化技术与β退火工艺，以提升抗应力腐蚀能力。

#### 3.2 表面防护方案
采用多层梯度复合涂层：
*   **底层**：环氧富锌底漆
*   **中间层**：聚苯墨烯改性环氧涂层
*   **面层**：氟碳树脂 (PVDF)
*   **特点**：综合电化学保护性能优秀。

#### 3.3 结构防腐蚀设计
*   **密封与排水**：蒙皮接缝处采用聚硫密封胶 (符合AMS3265标准)。
*   **电偶腐蚀防护**：异种金属接触面插入Ti-3Al-2.5V隔离垫片。
*   **目的**：有效提高结构连接处的抗腐蚀性能。

#### 3.4 局部强化方案
*   压气机叶片与进气口可根据要求进行特殊抗盐雾处理。
*   起落架可进行加强喷涂处理，提升耐磨损与耐腐蚀性能。

### 4. 主要系统与内部布置（SDR阶段示意）
*   **内部设备**：前至后大致包括雷达、航电综合处理机、弹舱、主油箱、发动机、进气道等。具体布置可参考总体布局示意图。
*   **发动机**：选型基于短距起降的高推力需求、高空推力衰减及隐身要求。

## 方案权衡与决策依据
对**方案一（单发）**与**方案二（双发）**的最终权衡如下：

| 考量维度 | 方案一 (单发) | 方案二 (双发) | 决策倾向 |
| :--- | :--- | :--- | :--- |
| **高速/快速响应** | 高速性能优秀 | 良好 | 倾向方案一 |
| **长航时/远距离侦察** | 结构紧凑，较省油 | 翼展长，载荷空间大 | 倾向方案一 |
| **成本控制** | 较低 | 较高 | 倾向方案一 |
| **隐身性能** | 一般 | **优秀**（融合体布局，可减小发动机口径） | **倾向方案二** |
| **体系化作战/冗余** | 一般 | **优秀**（双发提高控制与生存能力） | **倾向方案二** |
| **短距起降动力** | 受高空、高温削弱影响大 | **推力冗余高，满足需求** | **倾向方案二** |
| **抗气流干扰（大风）** | 相对较弱 | **稳定性与操控性更强** | **倾向方案二** |

**最终决策**：综合**短距起降的高推力需求、南海复杂环境（高温削弱、大风）、高空性能保持及隐身性要求**，**选择方案二（双发布局）**。
